SWE 205 In-Class Exercise
Spring 2021

Work with your group

The point of this exercise is for you to experience the web in the context of accessibility.

  1. Choose an alternate experience and install an appropriate browser extension. Three suggestions for Chrome:
    • ChromeVox (giving voice to chrome): You can use this to experience what it's like to navigate a site by sound instead of sight.
    • Colorblinding (simulates color-blindness). The color schemes on some pages don't work for color-blind users.
    • Navigating a web page without the use of a mouse. (No browser extension is necessary here.)
    The browser extensions install pretty fast. If you want to choose an alternate meaning for "different experience", that's ok too.
  2. Surf the web and look for problematic experiences.
Restriction: Pick web sites where usability is decent.
