SWE 619 Quiz Number 9
October 30, 2019

Name:
Express the following in CTL. Use the following declaration in constructing your answer. Specifically, the variable class can have exactly one of the given values at any one time. 
  VAR
     class  : {start, quiz, lecture, exercise, break, end};
  1. As soon as class starts, a quiz is always next.






    Answer: 
        class=start -> AX (class=quiz)
  2. If there is a quiz, then later there is a break.






    Answer: 
        class=quiz -> AF(class=break)
  3. After a break there is a never a quiz.






    Answer: 
        class=break -> AG (class!=quiz)
    Note, you could put an explicit delay in here (with an AX) to address the word "After", but, given that there is only one variable (ie "class"), and it can only have one value at any given time, there is no need to do so.
  4. If the quiz is ongoing in the current state, but over in some next state, then it is over in all next states.






    Answer: 
        class=quiz && [EX class!=quiz] -> AX [class!=quiz]
  5. Once we get to the end of class we don't do anything else.






    Answer: 
        class=end -> AG (class=end)
    If you think the word "Once" implies that we are guaranteed to get to the end of class, you could add this constraint as well: 
        AF( class = end)
    &
    class=end -> AG (class=end)
