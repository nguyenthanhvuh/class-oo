Consider the following implementation. 
public <T> static void rotate (List<T> list) {
   T x = list.remove(0);  
   list.add(x);          // places x at end of list
}
  1. Identify the behavior of the implementation in the following cases:
    • rotate(null); 

    • List <String> list = new ArrayList <String>();
rotate(list);


    • List <String> list = new ArrayList <String>();
list.add("cat");
rotate(list);


    • List <String> list = new ArrayList <String>();
list.add("cat");
list.add("dog");
rotate(list);

  2. Write preconditions that forbid the first two cases above.


  3. Transform each precondition into a postcondition with exception handling. Follow Bloch's guidelines.


  4. Change the implementation so that it satisfies the postconditions you just wrote. Again, follow Bloch's guidelines.
